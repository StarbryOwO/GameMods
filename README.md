## 中文说明

### 关于本仓库

本仓库用于整理、保存和发布我制作的游戏模组。

模组按照游戏名称分类存放。每个模组目录通常包含：

- 模组下载文件；
- 模组功能说明；
- 安装与使用教程；
- 兼容性说明；
- 已知问题及版本信息。

安装模组前，请进入对应模组的目录并阅读其中的 `README.md`。

**Player Secular Title as Head of Faith**

当真人玩家角色同时担任信仰领袖时，使角色继续显示符合原版条件的世俗统治者称谓，而不是教宗、牧首、哈里发等宗教领袖称谓。

该模组只调整称谓显示，不修改信仰领袖机制、政体、头衔归属、继承、AI 行为或其他游戏数值。

### 下载与安装

1. 在上方模组列表中选择需要的模组。
2. 下载对应的 ZIP 压缩包。
3. 解压压缩包。
4. 阅读模组目录中的 `README.md`。
5. 按照该模组的安装说明完成安装。
6. 在游戏启动器中启用模组。

不要直接将 ZIP 压缩包放入游戏的模组目录，除非对应模组的说明明确要求这样操作。

## English

### About This Repository

This repository is used to organize, preserve, and distribute game mods created by StarbryOwO.

Mods are organized by game. Each mod directory normally contains:

- the downloadable mod package;
- a description of the mod;
- installation and usage instructions;
- compatibility information;
- known issues and version details.

Before installing a mod, open its directory and read the included `README.md`.

### Mod List

#### Crusader Kings III

| Player Secular Title as Head of Faith | 1.0.0 | CK3 1.19.0.6 (Scribe) | [View Mod](./Crusader_Kings_III/Player_Secular_Title_as_Head_of_Faith/) · [Download ZIP](./Crusader_Kings_III/Player_Secular_Title_as_Head_of_Faith/Player_Secular_Title_as_Head_of_Faith.zip) |

**Player Secular Title as Head of Faith**

When a human-controlled ruler is also a head of faith, this mod keeps the appropriate vanilla secular ruler title instead of displaying a religious-head title such as Pope, Patriarch, or Caliph.

The mod only changes title display. It does not alter head-of-faith mechanics, government, title ownership, succession, AI behavior, or other gameplay values.

### Download and Installation

1. Select a mod from the list above.
2. Download its ZIP archive.
3. Extract the archive.
4. Read the `README.md` inside the mod directory.
5. Follow the installation instructions for that mod.
6. Enable the mod in the appropriate game launcher.

Do not place the ZIP archive directly into the game's mod directory unless the mod-specific instructions explicitly tell you to do so.

### Issues and Feedback

Bugs, compatibility problems, installation issues, and documentation errors can be reported through GitHub Issues:

[Open an Issue](https://github.com/StarbryOwO/GameMods/issues)

When reporting a problem, include:

- the game and game version;
- the mod and mod version;
- other enabled mods;
- the mod load order;
- steps to reproduce the problem;
- relevant logs or screenshots.
